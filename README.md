## Installing Sabit

To install **Sabit**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/Sabit-ecommerce-website.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/Sabit-ecommerce-website.git
```
